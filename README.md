# RyanGrimesBiography

This repository contains a public portfolio webpage for Ryan Grimes.

## Files included

- `index.html` — the webpage itself
- `styles.css` — the webpage styling
- `1745381769987.jpeg` — the portrait image shown on the page
- `RyanGrimes_Resume 2.pdf` — the resume linked from the page

## How to view the webpage

### Open it directly

1. Open the `Documents` folder on your computer.
2. Double-click `index.html`.
3. Your web browser will open the page.

## Notes for nontechnical users

- You do not need to install anything special to view the page if you open `index.html` directly.
- The resume PDF is linked from the page and can be opened from the browser.
- If you want to share the page publicly, it must be deployed to a web host like GitHub Pages or Netlify.
